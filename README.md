# Docker Multi-Container Project

## Technologies Used

- Docker
- Docker Compose
- Nginx
- Node.js
- MySQL
- Redis
- VMware Cloud Director

## Project Architecture

User → Frontend → Backend → MySQL & Redis

## Containers

- Frontend Container
- Backend Container
- MySQL Container
- Redis Container

## Features

- Multi-container deployment
- Docker Compose
- Persistent volumes
- Container networking
- Port mapping
